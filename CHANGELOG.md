# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-02-08

### Added
- Initial admin UI for the VibeSQL product family
- Vue 3 + TypeScript single-page application with Vite build tooling
- Express-based MCP (Model Context Protocol) server backend
- Unified administration hub for VibeSQL services
- npx/npm executable via `vibesql-admin` CLI entry point
